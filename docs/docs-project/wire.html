Wire

The Wire is an independent print and online music news magazine that covers alternative, underground and non-mainstream music releases and events while spotlighting artists through essays and interviews.

Web Design

Responsive Design

Redesign

This project was for a User Interface Design II class that required users to redesign a news style sight with formats for mobile, tablet, and desktop. This allowed me to practice responsive design techniques within programs like Figma.

TARGET AUDIENCE

The target user is defined as being someone between the ages of 20-30, living in urban areas, getting a degree of higher education or recently graduated.

The target user has an interest in obscure pieces of art or music that intellectually stimulates them. They wish to learn more in jazz, modern, indie, and experimental music specifically, as this magazine is targeted for those genres.

STYLE GUIDE

The color choices had to have some sort of sophistication, but still a more bright color pop, similar to the the magazine covers that they used. Both color palettes 2 and 3 were too bright and colorful for these sorts of text heavy pages, so the first color palette was the one I decided on. Also, the gold matched the high end standard that they hold themselves too.

FONTS

The font choices were designed to keep to that sophisticated style, along with a lot of copy text. The most readable font combination was style 2 and the font family had great cohesion.

ICONS

Lastly, the icons were styled with very minor differences, but the first option was chosen due to the drop shadow providing more clarity on the site.
