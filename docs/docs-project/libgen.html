Library Genesis

Library Genesis is a file sharing website for people to share educational materials such as textbooks and scientific articles. It is widely used amongst college students and those wishing to further their knowledge on a subject. It acts as a digital library in a sense.

Web Design

Responsive Design

Redesign

This project was for a user interface design class, the prompt being that I needed to redesign a website of choice. Simple enough. I had recently been to Library Genesis in order to get one of my textbooks and thought "Hey, this site kind of sucks!" So, I set out towards the redesign process.

TARGET AUDIENCE

The audience is targeted towards college students because they are often the ones searching for textbooks to avoid paying thousands of dollars on them over the years. The availability of scientific articles appeals to college students as well because it is a free alternative to paying for textbooks.

The target audience is reached as the site is well known amongst college students for its free resources. Students often discover the site through referral or social media posts, which lines up with the college student demographic as 70% are shown to be on Snapchat or Instagram and social interaction in college increases significantly increasing the odds that someone will get a referral.

GOALS

My immediate goals for the site are to clean up all of the excess information on the home page and in the search results while also providing a simple color palette/branding guide.

CARD SORT

The card sort activity was designed to help organize the site layout and provide some user input into what the target audience would want from the site. Luckily, my target audience was college students split 50% male, 50% female ranging from ages 18-24 allowing me to accurately test my audience through family and classmates.

TAKEAWAYS

The testing resulted in a surprisingly uniform result with 67% having made 3 categories. Reviewing over the participants submissions revealed that users wanted the site in categories based on subjects, Library Genesis’s cause, and file sharing.

WIREFRAMES

The home page was redesigned to greatly simplify and condense the amount of information, while still providing options to define the user’s search further. By having the logo centered above the search bar, the site is more reminiscent of a common search engine providing familiarity to the site. The menu was condensed into 3 options via the card sort clearing space for the search area.

The search results page was redesigned to have a more modern and blocky look, as the original site has very linear search results. This was intentionally to separate the different books, as if you were looking through a library.

STYLES

The fonts chosen were Merriweather Bold for the header text and Open Sans Light for the body copy. The color palette is also bright while offering a call back to the colors of the original logo.

CHANGES

In between the time between the final wireframes and the final prototype, some changes were made to improve upon the wireframes and the colors. Both the homepage and search results page had some flaws, so the first change made was to fix those.

THE BEFORE

The site before looked sketchy due to a lack of styling. The information is cluttered and it isn't clear what users should do first.

THE AFTER

The modern and sleek color palette welcomes users and has an immediate and clear goal on screen.

NEXT STEPS

The next steps for this project would be to prototype the site and test with users for feedback. The user feedback would lead into a cycle of feedback and changes until it met the needs of users.
